# V.I.N.C.E. — Virtually Intelligent Neural Cognitive Engine

Welcome to **V.I.N.C.E.**, a local, open-source AI assistant. 

> **⚠️ Note:** This is a very early version and is currently strictly **under development**. Expect bugs, missing features, and frequent changes!

## About the Author
V.I.N.C.E. is created and maintained by **Vince**, a 16-year-old developer from Hungary. I built this project to learn and create an intelligent assistant that runs locally on your machine.

## Features
* **Local LLM Integration:** Designed to hook directly into a local `llama.cpp` server.
* **Graphical Interface:** Built-in UI to chat with and configure V.I.N.C.E.
* **Tool Capabilities:** Supports online searching, web scraping, and system information retrieval.
* **Voice Integration:** Optional Text-to-Speech (TTS) and Push-to-Talk speech recognition capabilities.
